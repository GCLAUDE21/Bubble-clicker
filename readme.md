# BUBBLE CLICKER

Projet interactif réalisé dans le cadre de ma formation Développeur Web.

**Demo live**
[Voir la page](https://gclaude21.github.io/Bubble-Clicker/)

## Objectif

Créer un mini-jeu de bulles animées en JavaScript natif : des bulles apparaissent aléatoirement, le joueur doit cliquer dessus avant qu'elles disparaissent pour incrémenter son score.

## Compétences travaillées

- Manipulation du DOM avec JavaScript natif
- Génération aléatoire de propriétés CSS via JS
- Gestion des événements (click, timer)
- Animations CSS avec variables personnalisées

## Stack technique

JavaScript natif. HTML natif. CSS natif. Aucun framework ou bibliothèque externe.
